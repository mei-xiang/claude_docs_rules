# 爬虫策略

## 小区爬虫（安居客）

### 流程

1. 启动 Chromium 浏览器（有头/无头模式）
2. 加载已保存的 Cookie（绕过验证码）
3. 访问安居客列表页，提取小区基本信息
4. 检查验证码，需要时等待用户手动处理
5. 批量创建小区记录，保存当月价格
6. 逐个访问详情页，提取完整属性
7. 点击"小区解读"页签，提取特色和不足描述
8. 保存进度到 progress.json
9. 退出前强制保存 Cookie 和进度

### 反爬机制

- 安居客有强反爬机制（安全验证、登录要求）
- Playwright 模拟真实浏览器行为
- Cookie 持久化保存登录状态
- 验证码出现时暂停等待用户手动处理

### 已知问题

1. 首次运行需要手动处理验证码
2. 无头模式更易触发验证码
3. 列表页和详情页可能触发不同验证码

## 预售爬虫（深圳住建局）

### 流程

1. 访问首页获取 session cookie（无需浏览器）
2. 调用项目列表 API 获取所有预售项目
3. 对每个项目调用详情 API
4. 获取楼栋列表，调用楼栋详情 API
5. 获取单元字典，逐单元调用房间列表 API
6. 批量保存到后端数据库
7. 进度持久化到 presale-progress.json

### 技术要点

- 使用 axios 直接调用政府公开 API
- Node.js 18+ 需处理 SSL legacy renegotiation
- API 无反爬限制
- 支持断点续传（基于 passdate 时间过滤）
